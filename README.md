# 4-Bit-R2R-Resistor-Ladder-DAC
This 4-Bit R2R Resistor Ladder DAC Converter creates a circuit where you can create a analog signal using 4 digital outputs. Using a 5 volt system the maximum output voltage is 4.6875 V. This circuit uses multiple resistor voltage dividers to create "in-between" signals not possibly achieved by a digital output from a microcontroller.
